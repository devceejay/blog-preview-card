# Blog Preview Card

This project is a Blog preview card built with HTML and CSS

## Highlights:

- Utilized Flexbox to structure the card ensuring it adapts perfectly to different screen sizes
- Applied hover effects on the heading making it interactive and engaging to users
- Made good use of typography (font selection, line spacing and text alignment) to enhance readability
